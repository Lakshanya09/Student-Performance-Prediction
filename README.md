# 🎓 AI Student Performance Prediction System

The AI Student Performance Prediction System is a Machine Learning-based web application developed using Python, Streamlit, and a Random Forest Classifier. The application predicts a student's academic performance based on key academic and lifestyle factors such as attendance, study hours, sleep duration, screen time, stress level, assignment score, quiz score, class engagement, previous semester marks, weekend study hours, and commute time.

The system provides an interactive dashboard with visual analytics, prediction confidence, personalized AI recommendations, and downloadable performance reports. The project demonstrates the practical application of Machine Learning in the education domain to support data-driven academic decision-making.

## 🚀 Features

- 🎯 Student Performance Prediction using Random Forest
- 📊 Interactive Plotly Visualizations
- 🤖 AI-Based Personalized Recommendations
- 📈 Prediction Confidence Score
- 📄 Downloadable Student Performance Report
- 🎨 Modern and Responsive Streamlit Interface

## 🛠️ Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- Plotly
- Joblib
- ReportLab

## 📂 Machine Learning

- Algorithm: Random Forest Classifier
- Features Used: 11 Academic and Lifestyle Parameters
- Target: Student Performance Classification
